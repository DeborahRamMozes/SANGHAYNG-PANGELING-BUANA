# Carita Ratu Pakuan

Data pokok: Carita Ratu Pakuan nyebut balé watangan jeung balé tulis.

Harti panalungtikan: istilah ieu penting pikeun maca hubungan ruang, tulis, jeung hiasan visual dina lingkungan Sunda Kuna.

Tingkat sumber:

- A lamun naskah langsung dicekel.
- B/C lamun maké katalog atawa edisi.
- E lamun maké catetan web.

Watesan: fungsi ruangna kudu diverifikasi deui.
