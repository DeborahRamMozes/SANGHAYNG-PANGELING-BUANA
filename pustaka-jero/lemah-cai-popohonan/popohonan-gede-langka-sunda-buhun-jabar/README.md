# Popohonan Gede Langka Sunda Buhun Jabar

Folder ieu pikeun paket bacaan tabel popohonan gede/langka Sunda Buhun Jabar.

Asal bahan: tab PDF tina workbook `Tabel_Popohonan_Gede_Langka_Sunda_Buhun_Jabar.xlsm`.

## Folder

- `md/` versi Markdown supaya bisa dibaca langsung di GitHub.
- `csv/` versi CSV supaya tabel bisa dibuka langsung di GitHub.

## Tab

- Tabel Inti
- Pangeling Riksa
- Sumber
- Riksa Wanci
- Jalur Bibit
- Riksa Ngaran Lokal

## Posisi

Ieu asup ka `pustaka-jero/lemah-cai-popohonan/` sabab eusina ngeunaan pohon gede, leuweung, gunung, cai, lemah cai, wanci, jeung laku miara.
