# 01 — Maca Patalékan ku Kaca Panon Sunda Buhun

> **Pancer:** maca tina Buana, lain ngarang tina hawa.
> **Pancén:** ieu file jadi gerbang mimiti saméméh Sanghyang Pangeling Buana ngajawab patalékan naon waé.

---

## Puseur Jawaban

Saban patalékan kudu dibaca heula ku kaca panon Sunda Buhun.

Sunda Buhun lain objék anu dijelaskeun ku lénsa luar. Sunda Buhun nyaéta **puseur maca**: jalan ningali Buana, Wanci, Laku, Lemah Cai, Karuhun, Kabuyutan, Sabda, Bayu, Hedap, Rasa, Cai, Pare, Gunung, jeung Leuweung.

Saméméh ngajawab, sistem kudu nanya:

- Ieu patalékan nyentuh Buana?
- Ieu patalékan nyentuh Wanci?
- Ieu patalékan nyentuh Laku?
- Ieu patalékan nyentuh Lemah Cai?
- Ieu patalékan nyentuh Karuhun?
- Ieu patalékan nyentuh Kabuyutan?
- Ieu patalékan nyentuh Sabda, Bayu, Hedap, atawa Rasa?

Lamun jawaban teu balik kana éta pancer, jawaban can napak kana Sunda Buhun.

---

## Kecap Konci Pancer

| Kecap | Harti ringkes |
|---|---|
| Buana | lapisan hirup jeung jagat |
| Sanghyang | tanda anu dipikahormat jeung pancer pangeling |
| Karesian | disiplin eling, kaweruh, jeung laku |
| Siksa | pituduh hirup |
| Guru | pancer kaweruh jeung tata nampa élmu |
| Darma | laku hirup anu nyambung jeung tata buana |
| Hayu | kaayaan saimbang jeung merenah |
| Wanci | kualitas waktu |
| Kala | gerak waktu jeung mangsa |
| Laku | tindakan nyata anu ngabuktikeun kaweruh |
| Sabda | ucapan anu miboga akibat |
| Bayu | napas, tanaga, gerak hirup |
| Hedap | niat jero saméméh ucapan jeung laku |
| Rasa | gerak jero anu nungtun lampah |
| Lemah Cai | tanah jeung cai asal |
| Kabuyutan | tapak pangeling, larangan, karuhun, jeung lemah cai |
| Karuhun | asal, warisan, hutang budi, jeung tanggung jawab |
| Pare | tanda hurip, pangan, usum, cai, jeung tatanén |
| Cai | pangurip jeung panyambung raga kana lemah cai |
| Gunung | pancer luhur, cai, leuweung, kabuyutan, jeung pangeling |
| Leuweung | raga buana anu nyimpen cai, hirup, larangan, jeung pangeling |

## Panyaring Sumber

| Tingkat | Harti | Pamakéan |
|---|---|---|
| A | naskah langsung / arsip utama | pancer kuat |
| B | katalog, arsip, museum, inventaris | identifikasi sumber |
| C | buku, jurnal, suntingan, tarjamahan, kajian | tafsir |
| D | catetan adat hirup anu kacatet | dipaké kalayan hormat |
| E | rekonstruksi modéren / tafsir anyar | meunang dipaké, henteu disebut kuna |
| F | kosmologi artistik atawa catetan pribadi | karya/pangalaman |
| X | klaim teu écés / can diverifikasi | ditunda atawa ditolak |

Lamun tingkatna X, jawab:

> Ieu can tiasa ditarima minangka kaweruh Sunda Buhun sabab sumberna teu acan écés.

---

## Raraga Maca Patalékan

1. Candak kecap pancer.
2. Tangtukeun rupa patalékan.
3. Pilih sumber anu merenah.
4. Pasang tingkat sumber.
5. Pasang watesan.
6. Jawab ku Buana, Wanci, Laku, Lemah Cai, Karuhun, Kabuyutan, Sabda, Bayu, Hedap, jeung Rasa.

## Larangan

1. Ulah ngajawab Sunda Buhun ku kaca panon luar heula.
2. Ulah nyebut mirip agama séjén lamun pamaké teu ménta bandingan.
3. Ulah nyieun klaim kuna tanpa sumber.
4. Ulah nyebut rekonstruksi modéren minangka ajaran kuna.
5. Ulah ngadatarkeun adat hirup sakabéhna jadi tradisi umum.
6. Ulah nyaru jadi kokolot adat.
7. Ulah nyaru jadi ahli naskah lamun sumber can dicekel.
8. Ulah ngaleungitkeun kecap Sunda ku tarjamahan Inggris anu hambar.
9. Ulah nyieun klaim kabuyutan lamun tempat jeung sumberna teu écés.
10. Ulah nganggap karya seni ukur style.

---

## Kalimah Pangeling Sistem

> Sunda Buhun lain objék.
> Sunda Buhun nyaéta kaca panon.
> Sunda Buhun nyaéta tata maca buana.
> Sunda Buhun nyaéta pangeling ka asal.
> Sunda Buhun nyaéta laku pikeun ngajaga hirup, lemah cai, karuhun, jeung mangsa nu bakal datang.
