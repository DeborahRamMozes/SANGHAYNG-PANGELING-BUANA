# pustaka-jero

Folder ieu pikeun nyimpen sumber anu geus diberesihan jeung diverifikasi.

## Eusina
- naskah anu geus dipariksa
- transliterasi
- buku / jurnal anu geus jelas sumberna
- arsip museum anu geus aya katalogna
- glosarium Sunda Buhun anu geus diriksa
- kalénder Sunda anu boga rujukan écés
- pangucap suci anu boga sumber écés

## Status Sumber
Paké tanda kapastian:

| Tanda | Harti |
|---|---|
| A | Naskah langsung atawa arsip utama |
| B | Catetan arsip atawa museum |
| C | Tafsir jurnal atawa buku |
| D | Catetan adat hirup anu kacatet |
| E | Rekonstruksi modéren |
| F | Kosmologi artistik atawa catetan pribadi |
| X | Can diverifikasi / teu aya sumber écés |

## Pangeling
Sumber anu can pasti henteu diasupkeun ka dieu. Simpen heula dina `pustaka-luar/` atawa `riksa-sumber/`.
