# 02. Maca Wanci Poe Kala Sunda

## Fungsi Dokumen

Dokumen ieu pikeun maca wanci poé ngaliwatan Kala Sunda, kaayaan kiwari, jeung laku anu pantes pikeun raga, sabda, bayu, hedap, rasa, jeung eling.

## Aturan Maca Wanci

Wanci henteu dibaca salaku ramalan kosong. Wanci dibaca salaku hubungan antara:

- waktu,
- raga,
- cuaca,
- kabiasaan,
- lingkungan,
- rasa batin,
- jeung laku sapopoé.

Lamun perlu, data kiwari bisa dipaké: cuaca, kualitas hawa, kalender, musim, agenda sosial, jeung kaayaan lemah cai.

## Kerangka Wanci

```text
Tanggal:
Wanci:
Dasar Kala Sunda:
Data kiwari lamun aya:
Raga:
Sabda:
Bayu:
Hedap:
Rasa:
Eling:
Laku anu disarankeun:
Tingkat sumber:
Watesan:
```

## Tingkat Sumber

| Kode | Dipaké Pikeun |
|---|---|
| A | Naskah/arsip ngeunaan Kala Sunda. |
| B | Kajian akademik ngeunaan kalender, musim, budaya waktu. |
| C | Tradisi lisan ngeunaan wanci/laku. |
| D | Tafsir budaya jeung kuratorial. |
| E | Hipotesis praktis. |
| F | Ungkara pangeling sapopoé. |
| X | Ramalan liar tanpa dasar. |

## Catetan

Ulah ngajadikeun wanci salaku mesin takhayul. Wanci téh cara maca laku, lain alat pikeun nyieun sieun. Mesin sieun mah geus loba, biasana dijual ku algoritma.
