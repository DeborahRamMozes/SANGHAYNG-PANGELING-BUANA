# 07. Harti Cai, Pare, Gunung, Leuweung

## Fungsi Dokumen

Dokumen ieu pikeun maca unsur cai, pare, gunung, jeung leuweung dina kerangka Sunda Buhun, lemah cai, seni, laku sapopoé, jeung pangeling buana.

## Unsur Baca

### Cai
Cai dibaca salaku kahirupan, aliran, beberesih, ingetan, jeung hubungan antara raga jeung buana.

### Pare
Pare dibaca salaku pangan, hirup, bumi, daur, indung, jeung tata laku lemah cai.

### Gunung
Gunung dibaca salaku jangkungna, puseur, kabuyutan, wates, panjagaan, jeung arah eling.

### Leuweung
Leuweung dibaca salaku napas buana, panyimpen rahasia, panyalindung cai, jeung ruang hirup anu teu meunang diringkes jadi komoditas.

## Format Bacaan

```text
Unsur:
Konteks:
Harti literal:
Harti simbolik:
Harti dina laku:
Harti dina karya seni:
Sumber:
Tingkat sumber:
Watesan:
```

## Tingkat Sumber

| Kode | Dipaké Pikeun |
|---|---|
| A | Naskah/arsip primer. |
| B | Kajian budaya, lingkungan, sajarah, agraria. |
| C | Tradisi lisan jeung pangalaman komunitas. |
| D | Tafsir seni jeung budaya. |
| E | Hipotesis kerja. |
| F | Catetan puitik. |
| X | Klaim liar. |

## Catetan

Cai, pare, gunung, jeung leuweung lain hiasan kecap. Éta téh infrastruktur hirup. Lamun dirusak, anu ruksak lain simbol wungkul, tapi napas jeung beuteung.
