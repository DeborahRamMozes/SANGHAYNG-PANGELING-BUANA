# Ahung jeung Catetan Verifikasi

Dokumen ieu pikeun ngabenerkeun bacaan ngeunaan **Ahung**.

## Putusan Dasar

Ahung henteu meunang langsung ditolak lamun aya dina edisi naskah atawa catetan sumber. Statusna kudu dibaca ku protokol sumber:

- A lamun aya dina naskah/transliterasi langsung.
- B lamun aya dina katalog/metadata resmi.
- C lamun aya dina buku, edisi, tarjamahan, atawa kajian.
- E lamun jadi tafsir kiwari.
- X ngan lamun klaimna teu boga sumber.

## Catetan Kapastian

| Unsur | Putusan |
|---|---|
| Ahung salaku istilah anu disebut dina sumber | bisa valid lamun sumberna écés |
| Ahung salaku pangucap praktis | kudu ditahan lamun téks jeung konteksna can écés |
| Ahung salaku mantra karangan | ditolak lamun teu aya sumber |
| Ahung dina buku/edisi/kajian | C, bisa dipaké minangka rujukan kerja |
| Ahung dina naskah langsung | A, lamun faksimile/transliterasi dicekel |

## Watesan

Valid salaku istilah henteu sarua jeung bebas dipaké salaku pangucap. Pamakéan ritual, doa, mantra, atawa ucapan sakral kudu boga sumber téks jeung konteks.

## Putusan

Ahung kudu diasupkeun salaku catetan verifikasi, henteu dicerai-berai jadi file pondok anu teu ngajelaskeun statusna.
