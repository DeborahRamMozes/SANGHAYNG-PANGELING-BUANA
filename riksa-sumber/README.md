# riksa-sumber

Folder ieu pikeun panyaringan sumber, tingkat kapastian, jeung catetan verifikasi.

## Tanda Kapastian

| Tanda | Harti |
|---|---|
| A | Naskah langsung atawa arsip utama |
| B | Catetan arsip atawa museum |
| C | Tafsir jurnal atawa buku |
| D | Catetan adat hirup anu kacatet |
| E | Rekonstruksi modéren |
| F | Kosmologi artistik atawa catetan pribadi |
| X | Can diverifikasi / teu aya sumber écés |

## Format Riksa

```text
Judul sumber:
Jenis sumber:
Asal:
Pangarang / Penyusun:
Taun:
Tempat arsip:
Tumbu / nomor katalog:
Tingkat kapastian:
Status:
Catetan verifikasi:
Kaputusan:
```
