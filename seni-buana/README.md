# seni-buana

Folder ieu pikeun maca karya seni dumasar kana Buana, Rasa, Karuhun, Lemah Cai, warna, garis, raga, jeung lapisan.

## Eusina
- catetan karya seni
- tafsir warna
- tafsir garis
- tafsir raga
- hubungan karya jeung Buana
- hubungan karya jeung Karuhun
- hubungan karya jeung Lemah Cai
- hubungan karya jeung Rasa

## Format Catetan Karya

```text
Judul karya:
Seniman:
Taun:
Medium:
Ukuran:
Kecap konci:
Patali jeung Buana:
Patali jeung Rasa:
Patali jeung Karuhun:
Patali jeung Lemah Cai:
Catetan visual:
Catetan sumber:
```
