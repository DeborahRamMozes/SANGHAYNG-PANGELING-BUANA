# Istilah jeung Konsep Valid

Dokumen ieu ngaganti file bagian pondok anu matak ngabingungkeun.

## Istilah Utama

| Istilah | Status | Tingkat | Catetan |
|---|---|---|---|
| Sang-hyang | valid salaku istilah naskah | A/C | kudu dibaca dina kontéks sumber |
| Hyang | valid salaku istilah naskah | A/C | ulah dicampurkeun asal-asalan jeung istilah luar |
| Sanghyang | valid salaku bentuk gabungan | A/C | dipaké dina judul jeung istilah naskah |
| Siksa | valid salaku pituduh / ajaran / tata laku | A/C | baca dina SSKK jeung naskah patali |
| Karesian | valid salaku ranah resi / kaweruh / laku | A/C | henteu diringkes jadi moral umum |
| Guru | valid salaku pancer pangajaran | A/C | kudu dibaca dina konteks naskah |
| Sasana | valid salaku raraga pituduh | B/C | butuh rujukan edisi lamun dipaké rinci |
| Hayu / Rahayu | valid salaku konsep kasaimbangan | A/C/E | paké watesan lamun jadi tafsir kiwari |
| Darma | valid salaku jalan laku | A/C | baca dina Sewaka Darma jeung sumber patali |
| Kabuyutan | valid salaku tapak pangeling / tempat dihormat | A/C/D | perlu sumber tempat jeung komunitas |
| Karuhun | valid salaku asal jeung pangeling | A/C/D/E | ulah dipaké keur klaim tanpa sumber |
| Lemah Cai | valid salaku tanah-cai asal | C/D/E | patali jeung tempat hirup jeung tanggung jawab |

## Aturan Maca

1. Istilah valid henteu otomatis ngajadikeun unggal tafsir jadi valid.
2. Unggal istilah kudu boga sumber, konteks, jeung tingkat kapastian.
3. Lamun ngan tafsir kiwari, tandaan E.
4. Lamun catetan pribadi, tandaan F.
5. Lamun teu aya sumber, tandaan X.

## Putusan

Istilah dasar ieu kudu dibaca minangka sistem istilah, lain potongan leutik tanpa konteks.
