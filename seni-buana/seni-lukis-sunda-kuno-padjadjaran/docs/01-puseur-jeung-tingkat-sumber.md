# Puseur jeung Tingkat Sumber

Dokumen ieu bagian tina paket **Seni Lukis Sunda Kuna Jaman Pajajaran**.

## Puseur Jawaban

Aya dasar sumber pikeun nyebut yén dina tradisi tulis Sunda Kuna aya jejak kaweruh rupa, tulis, lukis, juru lukis, balé tulis, hiasan kadatwan, carita visual, jeung gambar kosmologis.

Bukti kerja ayeuna asalna tina:

1. Sanghyang Siksa Kandang Karesian.
2. Carita Ratu Pakuan.
3. Carita Parahyangan.
4. Falak Budha / Raffles Java 1.

Ieu henteu cukup pikeun nyieun rekonstruksi total gaya pasti. Ieu cukup pikeun nyusun peta data: istilah, ahli, motif, balé, tapak kadatwan, gambar naskah kosmologis, jeung jejak pasca-Pajajaran anu kudu dipariksa deui.
