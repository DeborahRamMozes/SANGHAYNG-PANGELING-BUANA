# Pancer Sanghyang Pangeling Buana

**Maca tina Buana, Wanci, Laku, Lemah Cai, Karuhun, Kabuyutan, Sabda, Bayu, Hedap, jeung Rasa**

Folder ieu jadi pancer pikeun maca deui Sanghyang Pangeling Buana tina panon Sunda Buhun jeung Sunda kiwari. Ieu lain tempat pikeun ngarang-ngarang pangucap kuna, lain mesin romantisasi, jeung lain panggung pikeun nyieun klaim kosong anu diasah siga keris tapi eusina karat.

## Prinsip Utama

1. Maca tina Buana, lain ngarang tina hawa.
2. Unggal klaim kudu dibéré tingkat sumber: A/B/C/D/E/F/X.
3. Pangucap Sunda Buhun kudu dibédakeun antara kutipan naskah, tafsir, rekonstruksi, jeung ciptaan anyar.
4. Wanci kudu dibaca ngaliwatan Kala Sunda jeung data kiwari lamun perlu.
5. Laku sapopoé kudu dibaca ngaliwatan raga, sabda, bayu, hedap, rasa, eling, jeung lemah cai.
6. Karya seni teu meunang direndahkeun jadi “style” wungkul. Karya seni kudu dibaca salaku tanda, rasa, ingetan, luka, karuhun, buana, jeung laku.

## Tingkat Sumber

| Kode | Tingkat Sumber | Harti |
|---|---|---|
| A | Naskah primer | Lontar, naskah kuna, prasasti, arsip primer, dokumen asli. |
| B | Kajian akademik kuat | Buku ilmiah, jurnal, tesis/disertasi, panalungtikan anu jelas metodena. |
| C | Tradisi lisan terverifikasi | Carita karuhun, wawancara, praktisi budaya, dicatet sumberna. |
| D | Tafsir kuratorial | Bacaan seni, semiotik, estetika, spiritual-kultural, kalawan wates jelas. |
| E | Hipotesis kerja | Pamadegan awal pikeun diuji deui. Teu meunang dianggap final. |
| F | Catetan kreatif | Puisi, pangeling, ungkara artistik, lain klaim sajarah. |
| X | Teu bisa dipaké | Klaim liar, sumber poék, cocokologi kasar, halusinasi mesin, atawa pangucap anu ngaku kuna tapi teu aya bukti. |

## Struktur

Tingali `SUMMARY.md` pikeun runtuyan kaca jeung dokumen dina folder ieu.

## Catetan Etik

Sanghyang Pangeling Buana kudu dijaga tina dua musuh: poho jeung ngarang-ngarang. Poho ngajadikeun urang leungit akar. Ngarang-ngarang ngajadikeun akar palsu dijual siga pusaka. Duanana sarua cilaka, ngan bajuna béda.
