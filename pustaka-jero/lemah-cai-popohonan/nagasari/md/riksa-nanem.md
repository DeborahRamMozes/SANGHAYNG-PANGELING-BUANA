# Riksa Nanem Nagasari

| Bagian | Cocog | Teu cocog |
|---|---|---|
| Lahan | Lembab, lega, aya cai, aya ruang tajuk. | Sempit, garing, drainase goreng. |
| Rawat | Aya cai, teduh, jeung pangropea. | Ditinggalkeun sanggeus melak. |
| Niat | Miara lila jeung sabar. | Hayang hasil instan. |
| Raga | Sanggup ngali, mulsa, naungan, nyiram. | Teu boga waktu jeung tanaga. |
| Catetan | Cocog dibaca tina lahan, cai, wanci, jeung rawat. | Ulah dijieun klaim lamun teu aya sumber. |
