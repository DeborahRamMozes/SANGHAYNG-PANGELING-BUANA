# 06. Catetan Arsip Istilah Ritual Naskah

## Fungsi

Dokumen ieu pikeun nyatet istilah, laku adat, naskah, kabuyutan, jeung rujukan anu aya hubunganana jeung Sanghyang Pangeling Buana.

## Aturan Catetan

Unggal catetan kudu misahkeun antara sumber, ringkesan, tafsir, jeung watesan. Ulah nyampur kutipan jeung pamadegan pribadi.

## Format

```text
Kode catetan:
Judul:
Jenis catetan:
Sumber:
Ringkesan:
Kutipan lamun aya:
Tafsir:
Tingkat sumber A/B/C/D/E/F/X:
Watesan:
Tindak lanjut:
```

## Kelas Sumber

- A: naskah atawa arsip primer.
- B: kajian akademik.
- C: tradisi lisan atawa wawancara.
- D: tafsir budaya.
- E: hipotesis kerja.
- F: catetan kreatif.
- X: teu bisa dipaké.

## Catetan

Arsip kudu ngajaga jejak. Lamun sumberna poék, tandaan poék. Ulah dipaké jadi dasar klaim.
