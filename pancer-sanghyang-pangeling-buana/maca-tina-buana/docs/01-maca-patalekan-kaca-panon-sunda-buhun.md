# 01. Maca Patalekan Kaca Panon Sunda Buhun

## Fungsi Dokumen

Dokumen ieu pikeun maca patalekan tina kaca panon Sunda Buhun: kumaha hiji pertanyaan dibuka, dipariksa, dikaitkeun jeung buana, karuhun, laku, rasa, jeung eling.

## Prinsip Maca

Patalekan teu cukup ngan ukur “naon jawabanana?” Patalekan kudu dibaca heula tina mana asalna:

- tina rasa,
- tina kabutuhan laku,
- tina luka sosial,
- tina lemah cai,
- tina karuhun,
- tina arsip,
- atawa tina kabingungan kiwari.

## Tingkat Sumber

| Bagian | Kode Sumber | Catetan |
|---|---|---|
| Kutipan langsung naskah | A | Wajib aya rujukan naskah/arsip. |
| Tafsir budaya | B/D | Dumasar kajian atawa bacaan kuratorial. |
| Tradisi lisan | C | Wajib dicatet narasumber jeung konteks. |
| Hipotesis | E | Kudu jelas yén ieu can final. |
| Ungkara kreatif | F | Teu meunang diklaim salaku naskah kuna. |
| Klaim poék | X | Ulah dipaké. |

## Format Catetan

```text
Patalekan:
Asal patalekan:
Kaca panon Sunda Buhun:
Kaca panon Sunda kiwari:
Tingkat sumber:
Watesan:
Catetan:
```

## Larangan

Ulah maksa unggal pertanyaan jadi mistik. Kadang pertanyaan téh ngan butuh sumber, logika, jeung saeutik rasa isin ka fakta.
