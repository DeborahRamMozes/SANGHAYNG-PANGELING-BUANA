# 04. Maca Karya Seni: Buana, Rasa, Karuhun, Lemah Cai

## Fungsi Dokumen

Dokumen ieu pikeun maca karya seni henteu salaku “style” wungkul, tapi salaku medan tanda anu nyimpen buana, rasa, karuhun, lemah cai, raga, luka, eling, jeung pangeling.

## Prinsip Maca Karya

Karya seni teu meunang direndahkeun jadi dekorasi visual. Dina kaca panon ieu, karya dibaca ngaliwatan:

- bahan,
- warna,
- raga bentuk,
- luka jeung bekas prosés,
- hubungan jeung lemah cai,
- ingetan karuhun,
- tanda cai, pare, gunung, leuweung,
- posisi awak jeung rasa,
- hubungan antara anu katingali jeung anu teu katingali.

## Tingkat Sumber

| Kode | Dipaké Pikeun |
|---|---|
| A | Dokumen asli karya, catetan seniman, arsip pameran. |
| B | Kajian seni/budaya anu kuat. |
| C | Wawancara seniman, tradisi lisan, saksi prosés. |
| D | Tafsir kuratorial. |
| E | Hipotesis bacaan karya. |
| F | Catetan puitik/artistik. |
| X | Klaim ngawur anu teu aya dasar. |

## Catetan

Karya seni lain budak leutik anu kudu diasupkeun kana kotak “gaya”. Karya seni téh raga anu mikir, luka anu nyarita, jeung arsip anu kadang leuwih jujur tibatan dokumen resmi.
