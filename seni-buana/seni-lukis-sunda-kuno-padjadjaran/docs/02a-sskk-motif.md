# SSKK: Motif Tulis

Data pokok: Sanghyang Siksa Kandang Karesian nyebut rupa-rupa tulis.

Istilah anu kudu dipariksa:

- pupunjengan
- hihinggulan
- kekembangan
- alas-alasan
- urang-urangan
- memetahan
- sisirangan
- taruk hata
- kembang tarate

Harti panalungtikan: ieu bukti yén aya kategori tulis jeung aya ahli rupa dina tradisi naskah Sunda Kuna.
