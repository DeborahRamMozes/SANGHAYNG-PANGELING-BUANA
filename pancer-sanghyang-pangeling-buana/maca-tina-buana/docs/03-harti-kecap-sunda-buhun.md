# 03. Harti Kecap Sunda Buhun

## Fungsi Dokumen

Dokumen ieu pikeun nyatet harti kecap Sunda Buhun, istilah naskah, istilah laku, jeung kecap anu perlu dibédakeun antara harti literal, harti simbolik, harti ritual, jeung harti kiwari.

## Aturan Utama

Ulah ngaku hiji kecap minangka kecap kuna lamun sumberna teu jelas. Lamun kecap téh ciptaan anyar, sebutkeun minangka ciptaan anyar. Basa henteu butuh dipalsukeun supaya katingali sakral. Éta téh kabiasaan pasar gelap intelektual.

## Format Entri

```text
Kecap:
Varian tulisan:
Harti literal:
Harti simbolik:
Harti ritual lamun aya:
Sumber:
Tingkat sumber A/B/C/D/E/F/X:
Catetan watesan:
Conto pamakéan:
```

## Kategori Harti

| Kategori | Harti |
|---|---|
| Literal | Harti dasar atawa harti kamus. |
| Simbolik | Harti dina bacaan budaya, seni, rasa, jeung tanda. |
| Ritual | Harti dina laku, upacara, kabuyutan, atawa tradisi. |
| Kiwari | Harti anu hirup dina pamakéan ayeuna. |
| Rekonstruksi | Harti anu dicoba dirangkai deui tina sumber. |
| Kreatif | Harti anyar pikeun karya seni atawa tulisan. |

## Catetan Sumber

Unggal kecap kudu boga catetan sumber. Lamun teu boga sumber, simpen heula salaku **E** atawa **F**, ulah dipaksa jadi **A**. Arsip lain tempat kosmetik pikeun ego.
